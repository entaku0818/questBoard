## 記事① ターゲットKW:「やりたいことリスト 書き方 30代」

### メタ情報

| 項目 | 内容 |
|---|---|
| ターゲットKW | やりたいことリスト 書き方 30代 |
| サブKW | やりたいことリスト テンプレ / 夢リスト 作り方 / 30代 人生設計 |
| 検索意図 | 「やりたいことリストを作りたいが、何を書けばいいかわからない30代」 |
| 想定文字数 | 4,000〜5,000字 |
| 想定読了時間 | 約10分 |

### タイトル案

**推奨**
```
やりたいことリストの書き方【30代向け】続けられる100リストの作り方とコツ
```

**サブ案**
```
【30代必見】やりたいことリストの書き方・続け方・叶え方を完全解説
```

### H2 構成（5本）

| # | H2見出し | 内容概要 | 文字数目安 |
|---|---|---|---|
| H2-1 | 30代こそやりたいことリストを作るべき理由 | 20代との違い・人生の有限性への気づき・後悔しない生き方の入口として | 600字 |
| H2-2 | やりたいことリストに「何を書くか」迷わない7つのジャンル | 旅行・仕事・家族・健康・お金・スキル・体験の7カテゴリ別に例示。空白を埋めるための問いかけを添える | 900字 |
| H2-3 | やりたいことリストが続かない3つの原因と解決策 | ①書いたまま忘れる→定期レビュー習慣 ②抽象的すぎる→クエスト化（行動レベルに分解）③一人で抱える→シェアで加速 | 800字 |
| H2-4 | 実際に書いてみよう：30代男性のやりたいことリスト例20選 | リアルな例20個を5カテゴリに分けて掲載。「あるある」感で共感→自分もやりたくなる流れ | 900字 |
| H2-5 | やりたいことリストをアプリで管理するメリット | 手帳・メモ帳との比較。通知・シェア・達成記録の3つの優位性。QuestBoardへの自然な誘導 | 600字 |

### CTA・内部リンク

- **記事末尾 CTA**: 「まずはQuestBoardでリストを作ってみよう（無料）」→ ストアリンク
- **内部リンク先**:
  - → 記事②「バケットリストアプリおすすめ」（比較検討層へのブリッジ）
  - → 記事③「人生設計アプリ 夢・目標管理」（人生設計に興味を持った読者向け）

### 構成上のポイント

- 冒頭に「30代の共感あるある」を入れて離脱を防ぐ
- H2-4 の例示は「自分も書けそう」と思わせる身近な粒度にする（「エベレスト登頂」ではなく「地元の山に子供と登る」レベル）
- H2-5 でQuestBoardに触れるが、あくまで選択肢の一つとして紹介し、ステマ感を排除する

## 記事② ターゲットKW:「バケットリスト アプリ おすすめ」

### メタ情報

| 項目 | 内容 |
|---|---|
| ターゲットKW | バケットリスト アプリ おすすめ |
| サブKW | バケツリスト アプリ / やりたいことリスト アプリ 無料 / 夢リスト 管理 アプリ |
| 検索意図 | 「バケットリストを管理するアプリを探しており、複数の選択肢を比較したい」 |
| 想定文字数 | 4,500〜5,500字 |
| 想定読了時間 | 約11分 |

### タイトル案

**推奨**
```
バケットリストアプリおすすめ5選【2024年版】無料・買い切り・機能を徹底比較
```

**サブ案**
```
【比較】バケットリスト管理アプリおすすめ5選｜やりたいことリストを続けるならこれ
```

### H2 構成（5本）

| # | H2見出し | 内容概要 | 文字数目安 |
|---|---|---|---|
| H2-1 | バケットリストアプリを選ぶ際の3つのチェックポイント | ①やりたいことリストとTODOを一緒に管理できるか ②共有・シェア機能があるか ③無料で十分使えるか | 600字 |
| H2-2 | バケットリストアプリおすすめ5選の比較一覧 | 表形式で5アプリを「機能・価格・UI・シェア機能・プラットフォーム」で比較。QuestBoardを含む | 800字 |
| H2-3 | 【1位】QuestBoard：やりたいことリスト×TODOを一つで管理 | 機能詳細・料金・シェアコードの使い方・向いている人を解説。スクショあり想定 | 900字 |
| H2-4 | 【2〜5位】他アプリの特徴と向いている人 | Notion・Todoist・Day One・標準メモアプリとの比較。それぞれの長所短所を公平に記述 | 900字 |
| H2-5 | バケットリストアプリを3ヶ月続けるためのコツ | ①週1回のレビュー習慣 ②友達とシェアして宣言効果 ③達成したら記念に記録する。継続率を上げる心理テク | 700字 |

### CTA・内部リンク

- **記事末尾 CTA**: 「QuestBoardを無料で試してみる」→ ストアリンク
- **内部リンク先**:
  - → 記事①「やりたいことリスト 書き方 30代」（まだリストを作っていない人向け）
  - → 記事③「人生設計アプリ 夢・目標管理」（さらに深く人生設計を考えたい人向け）

### 構成上のポイント

- H2-2 の比較表は「一番最初にスキャンされる」ので視認性を最優先
- 競合他社を公平に比較することで信頼性を高め、「QuestBoardが自分に合う人」を明確に絞り込む
- H2-3 はQuestBoardの一次情報（実際の操作感）を詳しく書くと、検索上位の比較記事に埋没しない差別化になる

## 記事③ ターゲットKW:「人生設計 アプリ 夢 目標管理」

### メタ情報

| 項目 | 内容 |
|---|---|
| ターゲットKW | 人生設計 アプリ 夢 目標管理 |
| サブKW | 人生設計 30代 / 目標管理 アプリ おすすめ / 夢 叶える 方法 アプリ |
| 検索意図 | 「人生を整理したい・夢と日常をつなげる目標管理の仕組みを探している30代」 |
| 想定文字数 | 5,000〜6,000字 |
| 想定読了時間 | 約13分 |

### タイトル案

**推奨**
```
人生設計に使えるアプリ5選【夢と目標管理を同時に】30代からの使い方ガイド
```

**サブ案**
```
【30代の人生設計】夢・目標管理をアプリで仕組み化する方法と厳選ツール5選
```

### H2 構成（5本）

| # | H2見出し | 内容概要 | 文字数目安 |
|---|---|---|---|
| H2-1 | 30代の人生設計で「夢と日常がバラバラ」になる理由 | 仕事・育児・日常タスクに追われて夢リストが別管理になる問題を整理。「夢は夢、仕事は仕事」の分断が夢の実現を遠ざける | 700字 |
| H2-2 | 人生設計アプリに必要な4つの機能 | ①夢・やりたいことリスト管理 ②日常TODOとの統合 ③進捗の可視化 ④他者へのシェア機能。各機能がなぜ重要かをエビデンスとともに説明 | 800字 |
| H2-3 | 夢を「クエスト化」する：目標を行動レベルに分解する方法 | 「海外移住したい」→「英語学習」「貯金計画」「ビザ情報収集」へのブレイクダウン手法。QuestBoardのクエスト機能とリンクさせる | 1,000字 |
| H2-4 | 人生設計・夢管理におすすめのアプリ5選 | QuestBoard・Notion・Habitica・Streaks・手帳アプリとの比較。「夢と日常の統合」という軸で評価 | 1,100字 |
| H2-5 | 人生設計アプリを使って夢を叶えた人がやっている3つの習慣 | ①月1回の人生棚卸し ②シェアコードで夢を宣言する ③小さな達成を記録して継続する。行動経済学の「コミットメント効果」にも言及 | 900字 |

### CTA・内部リンク

- **記事末尾 CTA**: 「QuestBoardで人生設計を始める（無料）」→ ストアリンク
- **内部リンク先**:
  - → 記事①「やりたいことリスト 書き方 30代」（まずリストの書き方を知りたい人向け）
  - → 記事②「バケットリストアプリおすすめ」（具体的なアプリを比較したい人向け）

### 構成上のポイント

- H2-3 は「クエスト化」というQuestBoard独自の概念を記事内で定義することで、ブランドワードとして育てる
- H2-5 に「コミットメント効果」などの心理学的根拠を入れると専門性が増し、シェアされやすくなる
- 記事全体を通じて「夢を持つのは恥ずかしくない・むしろ仕組み化すれば普通に叶う」というトーンを維持する

## 3記事の内部リンク構造まとめ

```
記事①「やりたいことリスト 書き方 30代」
  └→ 記事②（比較検討したい人）
  └→ 記事③（人生設計に踏み込みたい人）

記事②「バケットリスト アプリ おすすめ」
  └→ 記事①（リストの書き方に戻る）
  └→ 記事③（深く考えたい人）

記事③「人生設計 アプリ 夢 目標管理」
  └→ 記事①（書き方を知りたい人）
  └→ 記事②（アプリを比較したい人）
```

3記事が互いにリンクし合うことで、SEOの内部評価を高め、読者の回遊率を上げる。
すべての記事末尾にQuestBoardのストアリンクを設置し、インストールへの導線を統一する。
