## 説明文（本文）

### ▼ 最終版（4000文字以内、冒頭170文字が最重要）

```
「いつかやろう」が、いつまでも「いつか」のままになっていませんか？

QuestBoardは、あなたの「やりたいこと」と「日常のTODO」を一つのアプリで管理できる、行動管理ツールです。

─────────────────────
🗺️ QuestBoardでできること
─────────────────────

▶ やりたいことリスト（バケツリスト）
「富士山に登る」「家族で海外旅行」「副業で月5万」——漠然とした夢を、具体的なクエストに変えましょう。旅行・学習・体験・健康などカテゴリごとに整理でき、進捗状況（未着手／進行中／完了）も一目で把握できます。

▶ クエスト×TODO管理
やりたいことを「クエスト」として登録し、達成に向けた日常タスクを紐づけて管理。「この作業が、将来の自分につながっている」という実感が、継続の原動力になります。

▶ 目標ピラミッド
長期の夢から中期・短期の目標へとブレイクダウン。ピラミッド形式で人生の大目標を俯瞰しながら、日常のアクションに落とし込めます。

▶ シェアカードでXにポスト
自分のやりたいことリストと達成率をビジュアルカードに変換。スクリーンショットを撮ってXに投稿すれば、宣言することで実行力が上がります。達成数・進捗バー・カテゴリバッジ付きのおしゃれなカードが自動生成されます。

─────────────────────
💡 こんな人に選ばれています
─────────────────────

✅ 仕事は充実しているけど、自分の夢を後回しにしている
✅ Notionやメモ帳に夢リストを書いたけど、見返す習慣がない
✅ 友達と「いつかやろう」を言い合って、結局やっていない
✅ 人生の残り時間を意識し、やり残しをゼロにしたい
✅ シンプルで余計な機能がないTODOアプリを探している

─────────────────────
📱 シンプルで使いやすい設計
─────────────────────

・登録不要・即使える（アカウント作成不要）
・データはデバイスに保存（プライバシー安心）
・広告なし・シンプルなUI
・ダークグラデーションのSNS映えするシェアカード

─────────────────────

「いつか」を「今」に変える第一歩は、リストに書くことから。

QuestBoard — あなたの人生のミッションを、ここに。
```

## プロモーションテキスト（170文字・季節変更可）

```
来年こそやる——ではなく、今日リストに書こう。QuestBoardで「今年やること」を宣言。シェアカードをXに投稿して、お互いが刺激し合える新年のスタートを切ろう。

新年度は、やりたいことを全部書き出すところから。QuestBoardで夢リスト×日常TODOを一元管理。目標ピラミッドで長期の夢も短期アクションに落とし込める。
```
