# X（Twitter）投稿テンプレート — QuestBoard
## ターゲット: 30代男性・夢と日常をつなげたい層

---

## ■ テンプレート 1：共感訴求「いつかやろう、の墓場」

```
「いつかやろう」の墓場、持ってる人いると思う。

・富士山に登る
・海外に住んでみる
・副業で稼ぐ
・家族で長期旅行

全部、メモ帳に書いたっきり。

QuestBoardは、そのリストに「今日やること」を紐づけるアプリです。夢は計画になったとき、初めて現実になる。

まずリストに書くだけでいい。→ [リンク]

#やりたいことリスト #30代 #目標管理
```

**投稿タイミング推奨**: 日曜夜21〜23時（内省タイム）
**訴求ポイント**: 「あるある」から入り、解決策を提示。CTAは低ハードル。

---

## ■ テンプレート 2：シェアコード機能紹介「友達の夢を覗く」

```
QuestBoardの「シェアコード」が思ったより面白かった。

友達のやりたいことリストを送ってもらってインポートしたら、「え、こいつこんなこと考えてたの」って新鮮な発見が連発。

普段の会話じゃ出てこない、本音のやりたいこと。

自分のリストを送ったら「俺もそれやりたかった！」って返ってきた。

夢って、シェアすると叶いやすくなる気がする。→ [リンク]

#QuestBoard #やりたいことリスト #シェア
```

**投稿タイミング推奨**: 金曜夜・週末（友達との交流意識が上がる時間帯）
**訴求ポイント**: グロースループ機能を「体験談」風に自然に紹介。

---

## ■ テンプレート 3：哲学・刺さる訴求「残り時間の話」

```
30代って、人生の折り返し地点に気づく年代だと思う。

20代は「まだある」と思って先送り。
30代になって「あと何年ある？」って計算し始める。

やりたいことって、時間が経つほど遠くなる。体力、お金、状況——条件は変わり続けるから。

だから今、リストに書くことにした。書けば、動ける。

QuestBoard、無料です。→ [リンク]

#30代 #人生 #やりたいことリスト #目標
```

**投稿タイミング推奨**: 誕生日・年末年始・月初（節目に刺さる）
**訴求ポイント**: 「時間の有限性」という30代特有の感覚に訴求。無料であることを最後に添える。

## ■ 投稿戦略メモ

| テンプレ | 目的 | KPI |
|---|---|---|
| 1（共感） | 認知・共感獲得 | いいね・RT |
| 2（シェアコード） | 機能認知・バイラル | リプライ・インストール |
| 3（哲学） | 深い共感・ブランディング | ブックマーク・フォロワー増 |

### ハッシュタグ推奨セット
- `#やりたいことリスト #30代 #目標管理`（認知拡大向け）
- `#QuestBoard`（ブランドタグ、育てる）
- `#人生 #バケツリスト`（感情訴求向け）

### 投稿頻度推奨
- 週2〜3投稿（月水金 or 火木）
- 日曜夜は「内省系」投稿が刺さりやすい
