# X（Twitter）投稿テンプレート — QuestBoard
## ターゲット: 30代男性・夢と日常をつなげたい層

---

## ■ テンプレート 1：共感訴求「いつかやろう、の墓場」

```
「いつかやろう」の墓場、持ってる人いると思う。

・富士山に登る
・海外に住んでみる
・副業で稼ぐ
・家族で長期旅行

全部、メモ帳に書いたっきり。

QuestBoardは、そのリストに「今日やること」を紐づけるアプリです。夢は計画になったとき、初めて現実になる。

まずリストに書くだけでいい。→ [リンク]

#やりたいことリスト #30代 #目標管理
```

**投稿タイミング推奨**: 日曜夜21〜23時（内省タイム）
**訴求ポイント**: 「あるある」から入り、解決策を提示。CTAは低ハードル。

---

## ■ テンプレート 2：シェアカード機能紹介「宣言したら動ける」

```
QuestBoardの「シェアカード」を試したら、思った以上に良かった。

自分のやりたいことリストが達成率付きのカードになって、そのままXに投稿できる。

宣言することで、不思議と動ける。

「見てる人がいる」って意識が、先送りを減らすんだと思う。

無料です → [リンク]

#QuestBoard #やりたいことリスト #目標宣言
```

**投稿タイミング推奨**: 月曜朝7〜9時（週始め・目標意識が高い時間）
**訴求ポイント**: シェアカード機能を「体験談」風に自然に紹介。宣言→行動の心理を突く。

---

## ■ テンプレート 3：哲学・刺さる訴求「残り時間の話」

```
30代って、人生の折り返し地点に気づく年代だと思う。

20代は「まだある」と思って先送り。
30代になって「あと何年ある？」って計算し始める。

やりたいことって、時間が経つほど遠くなる。体力、お金、状況——条件は変わり続けるから。

だから今、リストに書くことにした。書けば、動ける。

QuestBoard、無料です。→ [リンク]

#30代 #人生 #やりたいことリスト #目標
```

**投稿タイミング推奨**: 誕生日・年末年始・月初（節目に刺さる）
**訴求ポイント**: 「時間の有限性」という30代特有の感覚に訴求。無料であることを最後に添える。

---

## ■ テンプレート 4：機能紹介「目標ピラミッド」

```
夢ってなぜ実現しないか、最近わかった気がする。

「でかい目標」と「今日のタスク」の間が、つながっていないから。

QuestBoardの目標ピラミッドは、人生の大目標→中期目標→今月やること、の3層で整理できる。

毎朝ピラミッドを見ると、今日何をすべきかが自然とわかる。

無料 → [リンク]

#目標管理 #やりたいことリスト #30代 #自己成長
```

**投稿タイミング推奨**: 月初・期初・年度始め（目標設定ムードが高いとき）
**訴求ポイント**: 「目標と行動のつながり」という課題を解決する機能として紹介。

---

## ■ テンプレート 5：ビフォーアフター「変化の実感」

```
3ヶ月前、「やりたいことリスト」をQuestBoardに書いた。

・温泉地を1人旅する
・読書を週2冊習慣にする
・筋トレを90日続ける

3ヶ月経って見返したら、3つとも達成してた。

書いてなかったら、たぶん1個もやってなかった。

リストに書く、それだけで人生が変わる気がしてる。

→ [リンク]

#QuestBoard #やりたいことリスト #達成 #習慣
```

**投稿タイミング推奨**: 季節の変わり目（3月・6月・9月・12月）
**訴求ポイント**: 具体的な達成事例で「自分もできる」という自己効力感を刺激。

## ■ テンプレート 6：ユーモア訴求「Notionで夢リストを管理していた話」

```
Notionで「人生でやりたいことリスト」を作った。

凝りすぎてタグ・期限・優先度・カバー画像まで設定した。

完成した達成感で満足して、2度と開かなかった。

QuestBoard、シンプルすぎるくらいシンプルです。それがちょうどいい。→ [リンク]

#Notion #やりたいことリスト #あるある
```

**投稿タイミング推奨**: 平日昼（息抜きタイム）・ユーモア枠
**訴求ポイント**: 「凝りすぎアプリ」との差別化。Notionユーザーをターゲットに刺さる自虐ネタ。

---

## ■ テンプレート 7：数字訴求「やりたいことを書くだけで実現率が上がる」

```
「目標を紙に書いた人は、書かなかった人より42%多く達成した」

という研究を読んで、素直に試した。

・転職する → した
・一眼レフを買う → 買った
・沖縄に行く → 行った

書くって、本当に効く。

やりたいことリスト、QuestBoardに書いてみて。無料。→ [リンク]

#やりたいことリスト #目標達成 #習慣化 #30代
```

**投稿タイミング推奨**: 日曜夜・月曜朝（自己啓発モードの高い時間）
**訴求ポイント**: 研究データ引用でロジック訴求。実体験風ビフォーアフターで信頼感を付加。

## ■ 投稿戦略メモ

| テンプレ | 目的 | KPI |
|---|---|---|
| 1（共感） | 認知・共感獲得 | いいね・RT |
| 2（シェアカード） | 機能認知・バイラル | リプライ・インストール |
| 3（哲学） | 深い共感・ブランディング | ブックマーク・フォロワー増 |
| 4（ピラミッド） | 機能認知・差別化 | クリック・インストール |
| 5（ビフォーアフター） | 信頼獲得・CVR向上 | 保存・インストール |
| 6（ユーモア） | 拡散・新規認知 | RT・いいね |
| 7（数字） | 権威付け・行動喚起 | クリック・保存 |

### ハッシュタグ推奨セット
- `#やりたいことリスト #30代 #目標管理`（認知拡大向け）
- `#QuestBoard`（ブランドタグ、育てる）
- `#人生 #バケツリスト`（感情訴求向け）
- `#目標達成 #習慣化`（自己啓発層向け）

### 投稿頻度推奨
- 週2〜3投稿（月水金 or 火木）
- 日曜夜は「内省系」投稿が刺さりやすい
- 月初・年度始めは「目標設定系」を優先
- 平日昼12時はユーモア・あるある系を投入

### 優先投稿順（ローンチ直後）
1. テンプレ1（認知獲得）→ 2. テンプレ7（信頼付け）→ 3. テンプレ2（機能紹介）→ 4. テンプレ6（拡散）
